# İşletim Sistemleri - CPU Zamanlama Simülasyonu
**Hazırlayan:** Cenker Sipahi

Bu proje; FCFS, SJF, Round Robin ve Priority CPU zamanlama algoritmalarını simüle eder.

## Nasıl Çalıştırılır?
1. Projeyi bilgisayarınıza indirin.
2. Terminali açın.
3. `dotnet run` komutunu yazın.
4. Program otomatik olarak `case1.txt` ve `case2.txt` dosyalarını okuyup raporları oluşturacaktır.

## Çıktılar
Sonuçlar `Sonuclar_Case1` ve `Sonuclar_Case2` klasörlerinde `.txt` formatında saklanır.
